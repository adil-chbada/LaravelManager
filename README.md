<br>
<h2>after install go to  this  URL :<b> http://localhost:8000/lManager </b></h2>
<b>NOTE</b>
<p>If you want to execute "php artisan clear:cache", use :<b> http://localhost:8000/clear </b> that helps resolve many issues	</p>
